สำรวจโลกของระบบสมองกลฝังตัว

เอกสารประกอบการสอน (รูปแบบ A4 แนวนอน)

ระบบสมองกลฝังตัวคืออะไร?

รากฐานของอุปกรณ์อัจฉริยะทุกชนิดรอบตัวเรา

นิยามและหัวใจสำคัญ

ระบบสมองกลฝังตัว (Embedded System) คือระบบคอมพิวเตอร์ที่ถูกออกแบบมาเพื่อ **ทำงานเฉพาะทาง** โดยเป็นการผสานรวมกันระหว่าง **ฮาร์ดแวร์** และ **ซอฟต์แวร์** เพื่อทำหน้าที่ควบคุม ตรวจสอบ หรือตอบสนองต่อสภาพแวดล้อมภายนอก มันเป็น "สมอง" ที่ซ่อนอยู่เบื้องหลังการทำงานของอุปกรณ์ต่างๆ ทำให้เกิดความ "ฉลาด" และทำงานได้เองโดยอัตโนมัติ

คุณลักษณะเด่น

ทำงานเฉพาะด้าน (Single Purpose)

ถูกออกแบบมาเพื่องานเดียว เช่น ควบคุมอุณหภูมิในเครื่องปรับอากาศ หรือจัดการระบบเบรก ABS ในรถยนต์

ทำงานได้เอง (Autonomous)

สามารถทำงานได้ด้วยตัวเองตามที่โปรแกรมไว้ โดยไม่ต้องมีการควบคุมจากมนุษย์ตลอดเวลา และมักทำงานแบบ Real-time

เชื่อถือได้สูง (High Reliability)

ต้องทำงานได้อย่างถูกต้องและต่อเนื่อง เพราะความผิดพลาดอาจนำไปสู่ความเสียหายร้ายแรงได้

ประหยัดพลังงาน (Energy Efficient)

มักใช้พลังงานต่ำเพื่อให้ทำงานกับแบตเตอรี่ได้เป็นเวลานาน

ต้นทุนต่ำ (Low Cost)

เนื่องจากผลิตในปริมาณมาก ต้นทุนต่อหน่วยจึงต้องต่ำเพื่อให้ผลิตภัณฑ์สามารถแข่งขันในตลาดได้

ขนาดเล็ก (Compact Size)

ต้องมีขนาดเล็กพอที่จะ "ฝัง" เข้าไปในอุปกรณ์หรือระบบที่ใหญ่กว่าได้

6 โดเมนของพลังงานทางกายภาพ

วิธีที่ระบบสมองกลฝังตัวรับรู้และโต้ตอบกับโลกทางกายภาพ

ระบบสมองกลฝังตัวทำงานโดยการแปลงพลังงานจากโดเมนหนึ่งไปอีกโดเมนหนึ่งผ่านเซ็นเซอร์และแอคทูเอเตอร์ โดเมนพลังงานหลัก 6 ประเภทที่เกี่ยวข้องกับการทำงานของระบบเหล่านี้ได้แก่:

1. พลังงานรังสี (Radiant)

เกี่ยวข้องกับแสง (Light), คลื่นแม่เหล็กไฟฟ้า (Electromagnetic Waves) เช่น เซ็นเซอร์แสง, กล้อง

2. พลังงานกล (Mechanical)

เกี่ยวข้องกับการเคลื่อนที่ (Motion), แรง (Force), ความดัน (Pressure) เช่น เซ็นเซอร์ความเร่ง, มอเตอร์

3. พลังงานความร้อน (Thermal)

เกี่ยวข้องกับความร้อน (Heat), อุณหภูมิ (Temperature) เช่น เซ็นเซอร์อุณหภูมิ, ฮีตเตอร์

4. พลังงานไฟฟ้า (Electrical)

เกี่ยวข้องกับแรงดันไฟฟ้า (Voltage), กระแสไฟฟ้า (Current) ซึ่งเป็นพื้นฐานการทำงานของวงจร

5. พลังงานเคมี (Chemical)

เกี่ยวข้องกับองค์ประกอบทางเคมี (Composition), ปฏิกิริยา (Reactions) เช่น เซ็นเซอร์แก๊ส, แบตเตอรี่

6. พลังงานแม่เหล็ก (Magnetic)

เกี่ยวข้องกับสนามแม่เหล็ก (Magnetic Fields) เช่น เซ็นเซอร์เข็มทิศ, มอเตอร์บางชนิด

ส่วนประกอบหลักของระบบ

องค์ประกอบสำคัญที่ทำงานร่วมกันเพื่อสร้างระบบสมองกลฝังตัวที่สมบูรณ์

🧠 โพรเซสเซอร์: สมองของระบบ

หน่วยประมวลผลกลางที่ทำหน้าที่คำนวณและตัดสินใจตามโปรแกรมที่กำหนดไว้ มี 2 ประเภทหลักที่ใช้ในระบบสมองกลฝังตัว

ไมโครโปรเซสเซอร์ (μP)

เป็นแค่ CPU ต้องต่อพ่วงกับ RAM, ROM และ I/O ภายนอก

  • ✔ มีความยืดหยุ่นสูงในการออกแบบ
  • ✔ ประสิทธิภาพสูง เหมาะกับงานซับซ้อน
  • ✘ ขนาดใหญ่และซับซ้อนกว่า
  • ✘ ใช้พลังงานมากกว่า

ไมโครคอนโทรลเลอร์ (μC)

รวม CPU, RAM, ROM, I/O ไว้ในชิปเดียว (System-on-a-Chip)

  • ✔ ขนาดเล็กและต้นทุนต่ำ
  • ✔ ประหยัดพลังงาน
  • ✔ ออกแบบง่ายและรวดเร็ว
  • ✘ ประสิทธิภาพและความจุจำกัด

💾 หน่วยความจำ (Memory)

ส่วนสำคัญสำหรับจัดเก็บโปรแกรมและข้อมูลที่ใช้ในการทำงานของระบบสมองกลฝังตัว

ROM (Read-Only Memory)

หน่วยความจำแบบอ่านอย่างเดียว ใช้เก็บโปรแกรม (Firmware) ที่ไม่เปลี่ยนแปลง

  • ✔ ข้อมูลไม่หายเมื่อปิดเครื่อง
  • ✔ เหมาะสำหรับเก็บโปรแกรมหลัก
  • ✘ เขียนข้อมูลซ้ำได้จำกัด (หรือไม่ได้เลย)

RAM (Random Access Memory)

หน่วยความจำแบบเข้าถึงโดยสุ่ม ใช้เก็บข้อมูลชั่วคราวที่กำลังประมวลผล

  • ✔ เขียนและอ่านข้อมูลได้รวดเร็ว
  • ✔ เหมาะสำหรับเก็บข้อมูลที่เปลี่ยนแปลงบ่อย
  • ✘ ข้อมูลหายเมื่อปิดเครื่อง

👁️ อินพุต: การรับรู้จากโลกภายนอก

เซ็นเซอร์ทำหน้าที่แปลงค่าทางกายภาพ (เช่น อุณหภูมิ, แสง, การเคลื่อนไหว) ให้เป็นสัญญาณไฟฟ้าเพื่อให้ระบบประมวลผลได้

🦾 เอาต์พุต: การกระทำต่อโลกภายนอก

แอคทูเอเตอร์ทำหน้าที่แปลงสัญญาณไฟฟ้ากลับเป็นพลังงานกลหรือการกระทำทางกายภาพ เพื่อควบคุมอุปกรณ์ต่างๆ

สถาปัตยกรรมและซอฟต์แวร์

โครงสร้างพื้นฐานและโปรแกรมที่ขับเคลื่อนระบบให้ทำงานตามเป้าหมาย

สถาปัตยกรรมคอมพิวเตอร์

Von Neumann

เก็บโปรแกรมและข้อมูลในหน่วยความจำเดียวกัน ใช้บัสร่วมกัน

CPU ↔️ [BUS] ↔️ [MEMORY (Program + Data)]

เรียบง่าย แต่เกิดปัญหาคอขวด (bottleneck) เพราะต้องเข้าถึงหน่วยความจำทีละอย่าง

Harvard

แยกหน่วยความจำสำหรับโปรแกรมและข้อมูลออกจากกัน มีบัสแยกกัน

CPU ↔️ [PROG BUS] ↔️ [PROGRAM MEMORY]
CPU ↔️ [DATA BUS] ↔️ [DATA MEMORY]

ทำงานเร็วกว่าเพราะเข้าถึงโปรแกรมและข้อมูลได้พร้อมกัน นิยมในไมโครคอนโทรลเลอร์

รูปแบบซอฟต์แวร์ในระบบสมองกลฝังตัว

ซอฟต์แวร์หรือ "เฟิร์มแวร์" มีหลายรูปแบบ ขึ้นอยู่กับความซับซ้อนและความต้องการในการตอบสนองของระบบ

Simple Control Loop

ซอฟต์แวร์ทำงานในลูปที่ไม่มีวันสิ้นสุด เหมาะสำหรับงานควบคุมที่ไม่ซับซ้อน

Interrupt-controlled System

ระบบจะอยู่ในสภาวะพักและถูกเรียกให้ทำงานเมื่อมีเหตุการณ์ที่กำหนด (เช่น Timer, External Interrupt)

Cooperative Multitasking

ทำงานคล้ายวนลูปแต่เพิ่ม Task ได้ง่ายกว่า โดย Task จะต้องคืนการควบคุมเอง

Preemptive Multitasking (RTOS)

มักพัฒนาเป็นระบบปฏิบัติการแบบเรียลไทม์ (RTOS) มีการจัดสรรเวลาทำงานของ Task โดยระบบ

ตัวอย่างการใช้งานในชีวิตประจำวัน

ระบบสมองกลฝังตัวซ่อนอยู่ในอุปกรณ์มากมายรอบตัวเรา

รถยนต์

รถยนต์สมัยใหม่มีระบบสมองกลฝังตัว (ECU) หลายสิบตัว ทำหน้าที่ควบคุมทุกอย่าง

เครื่องซักผ้า

ไมโครคอนโทรลเลอร์ควบคุมมอเตอร์, วาล์วน้ำ, และฮีตเตอร์ให้ทำงานตามขั้นตอน

เตาไมโครเวฟ

ระบบสมองกลฝังตัวทำหน้าที่อ่านคำสั่ง, ควบคุมเวลา, และกำลังวัตต์ของคลื่นไมโครเวฟ

กล้องดิจิทัล

มีระบบประมวลผลภาพที่ซับซ้อนเพื่อจัดการข้อมูลจากเซ็นเซอร์รับภาพและปรับโฟกัส

เครื่องวัดความดัน

เซ็นเซอร์วัดความดันส่งข้อมูลไปยังไมโครคอนโทรลเลอร์เพื่อคำนวณและแสดงผล

เครื่องพิมพ์

รับข้อมูลจากคอมพิวเตอร์, แปลงเป็นคำสั่ง, และควบคุมการเคลื่อนที่ของหัวพิมพ์

เครื่องปรับอากาศ

ใช้เซ็นเซอร์อุณหภูมิในการตรวจสอบและควบคุมการทำงานของคอมเพรสเซอร์

สมาร์ทโฟน

เป็นตัวอย่างของระบบสมองกลฝังตัวที่ซับซ้อนที่สุด ประกอบด้วยโปรเซสเซอร์หลักและร่วม

~1769: ยุคปฏิวัติอุตสาหกรรม

1971: กำเนิดไมโครโปรเซสเซอร์
